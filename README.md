# 🎞️ Image Slider / Carousel

A modern, responsive **Image Slider (Carousel)** built using **HTML, CSS, and JavaScript**.
It features smooth transitions, thumbnails, navigation controls, and a sleek UI design.

---

## ✨ Features

* 🔄 Smooth slide transitions with animations
* ⏭️ Next / Previous navigation buttons
* 🔘 Interactive dots indicator
* 🖼️ Clickable thumbnail previews
* 🔢 Slide counter (e.g., `1 / 6`)
* 🎨 Modern UI with glassmorphism & gradient effects
* 📱 Fully responsive design
* ⚡ Lightweight (no external JS libraries)

## 🛠️ Technologies Used

---

## 🎮 How It Works

* Slides are controlled using JavaScript by toggling `active` and `exit` classes.
* Navigation:

  * **Buttons** → Move next/previous
  * **Dots** → Jump to specific slide
  * **Thumbnails** → Preview & select slide
* Dynamic elements (dots + thumbnails) are generated automatically.
